# Anthony Leuterio — Global Real Estate Visionary

A premium, high-fidelity abstract portfolio showcasing the strategic ventures and global recognition of Anthony Gerard Leuterio, founder of Filipino Homes and Leuterio Realty.

## Core Ecosystem
- **Leuterio Realty:** The nationwide flagship brokerage powerhouse.
- **Filipino Homes:** A massive digital PropTech ecosystem.
- **Rent.ph:** The significant rental marketplace infrastructure in the Philippines.

## Technologies Used
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform) for high-end cinematic sequences
- ScrollTrigger for immersive exploration

## Features
- **Cinematic Experience:** Fluid animations and atmospheric shifts.
- **Sovereign UI:** A high-contrast, regal aesthetic inspired by global leadership.
- **Dynamic Themes:** seamless transitions between Obsidian (Dark) and Elite (Light) modes.
